Install and Setup
====================

Requirements
----------------

BeGin works with the following Python packages:

- pytorch >= 1.8
- dgl >= 0.6.1
- ogb >= 1.3
- torch-scatter >= 2.0.6
- torch-sparse >= 0.6.9

For running some examples, you may need the following additional packages:

- quadprog
- cvxpy
- qpth

Installation
----------------

You can install BeGin using pip with the following command:

.. code:: bash

    pip install begin-cl
