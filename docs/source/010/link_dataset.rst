-----
BitCoin-OTC
-----
The dataset `BitCoin-OTC <https://snap.stanford.edu/data/soc-sign-bitcoin-otc.html>`_ is a who-trust-whom network.

We support `Task-`, `Class-` and `Time-Incremental` settings for this dataset.
