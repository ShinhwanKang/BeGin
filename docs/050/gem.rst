GEM
============

Gradient Episodic Memory (GEM) is a replay-based continual learning method which stores data from previous tasks and prevents the increase of losses on them while learning a new task.
For the details, see the `original paper <https://arxiv.org/pdf/1706.08840.pdf>`_.
